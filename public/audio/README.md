## Audio Format

- Format: MP3
- Recommended bitrate: 128 kbps or higher
- Sample rate: 44.1 kHz
- Ensure audio quality is clear for educational purposes

## Note

The audio synchronization feature works by calculating word timing based on audio duration. For more precise synchronization, you may want to implement word-level timing data in the future.
